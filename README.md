# Sewing

A structured, interactive, offline-capable learning site that teaches sewing from absolute beginner to confident home sewer.

## What's Inside

- **10 step-by-step lessons** from tools and setup through your first complete project
- **4 knowledge quizzes** to test what you've learned
- **4 printable reference pages**: stitch glossary, fabric guide, needle chart, common problems
- **Progress tracking** saved in your browser — no account needed
- **Works completely offline** — just open `index.html`

## How to Use

1. Open `sewing/index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)
2. No internet connection required — everything is local
3. Work through lessons in order — each one builds on the last
4. Mark lessons complete as you go; your progress is saved automatically
5. Take quizzes to test your knowledge
6. Use the reference pages as printable cheat sheets

## Technical Details

- Pure HTML, CSS, and JavaScript — no frameworks, no build step
- Progress stored in browser localStorage
- Works on `file://` protocol (open directly without a server)
- Compatible with Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- Targets WCAG 2.2 Level AA accessibility

## Exporting Your Progress

Click **Export Progress** in the footer to download your progress as a `sewing-progress.json` file. Use **Import Progress** to restore it on another browser or computer.
